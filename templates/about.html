My Project: PDF Merger

The PDF Merger web app is a result of my dedication and passion for creating practical solutions. This application allows users to merge multiple PDF files into a single document, providing convenience and efficiency in managing PDF files. I have implemented a user-friendly interface and utilized various web development technologies to ensure a seamless user experience.

Technologies Used

In developing the PDF Merger web app, I utilized Python as the backend programming language and Flask as the web framework. HTML and CSS were used for creating the user interface, while JavaScript added interactivity and enhanced functionality. I also leveraged additional libraries and tools to handle file uploads, manage PDF merging, and handle user requests.
